# Diseño en Ingenieria
__Pagina web de voluntariados y hospedajes__
*Este proyecto presenta el desarrollo de una plataforma humanitaria para voluntariados que facilita el acceso a hospedajes y oportunidades laborales para voluntarios. Inspirada en el modelo de Trivago, la plataforma busca optimizar la experiencia del usuario mediante un diseño centrado en sus necesidades para garantizar accesibilidad e inclusión.*
